# Гаухар қыз ұзату шақыру сайты

Бұл сайт статикалық HTML/CSS/JS түрінде жасалған. Deploy жасау үшін барлық файлдар осы папкада дайын.

## Netlify арқылы

1. https://app.netlify.com/drop сайтына кіріңіз.
2. Осы папканы толық сүйреп апарып тастаңыз: `C:\Users\aibol\OneDrive\Documents\гаухар`
3. Netlify сайт сілтемесін өзі береді.

## Vercel арқылы

1. https://vercel.com/new сайтына кіріңіз.
2. Осы жобаны GitHub-қа жүктеп, Vercel-ге қосыңыз.
3. Framework ретінде `Other` таңдасаңыз болады.
4. Build command бос, Output directory бос немесе `.`.

## Негізгі файл

Сайттың басты файлы: `index.html`
